# FTC WI.R.E.S.

**Wisconsin Rises to Enable STEM Growth**

FTC WI.R.E.S. is a community resource page for FIRST Tech Challenge teams, led by Team 13201 Hazmat with contributions from teams, mentors, and alumni across Wisconsin.

We build guides, examples, and templates for common FTC tools so newer teams can get up to speed faster — and experienced teams can keep learning.
